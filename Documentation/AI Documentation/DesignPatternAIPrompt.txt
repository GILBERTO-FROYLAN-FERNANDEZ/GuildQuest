## **What is the State Pattern?**

The State pattern allows an object to alter its behavior when its internal state changes. The object will appear to change its class. It's particularly useful when:
- An object's behavior depends on its state
- Operations have large, multi-part conditional statements that depend on the object's state
- State-specific behavior should be defined independently

---
